# skills

Agent skills for iOS development, structured after [mattpocock/skills](https://github.com/mattpocock/skills).

## Install (Claude Code)

As a plugin — works on any machine:

```
/plugin marketplace add yumin-jung/skills
/plugin install yumin-jung-skills@yumin-jung
```

Or clone and symlink into your local skill directories:

```bash
git clone https://github.com/yumin-jung/skills.git
cd skills && ./scripts/link-skills.sh
```

## Skills

| Skill | Description |
| --- | --- |
| [computer-use-ios](./skills/ios/computer-use-ios/SKILL.md) | Control the iOS Simulator headlessly via `simctl` + `idb` — screenshots, tap, swipe, text — instead of driving the real mouse with a computer-use tool. |
